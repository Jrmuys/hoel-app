# MVP Scope (Phase 1)

## In Scope

- Backend service foundation in Go
- SQLite schema and migration runner
- Integration error logging and health metadata
- PGH polling and cache-backed read endpoint
- TickTick OAuth, daily task sync, and task complete/uncomplete mutation
- Frontend shell with Status Bar and Daily Operations modules
- Telegram alerting for repeated integration failures

## Deferred

- Home Assistant integration
- Calendar integration
- TickTick shopping and maintenance modules
- Rich TickTick CRUD beyond complete/uncomplete
- Push/webhook-based sync
